## Mathematics for neuroscience and learning

I am a 21yo _aspiring_ mathematician, currently studying at ENSIMAG - UGA in Grenoble, France.
Interested in uncovering the secrets of the brain through statistics and geometry : Neuroscience, Machine Learning, Geometric Learning and more.

If you are interested in any of my work, feel free to contact me at mhacini.pro@gmail.com.

Current public work includes : 

- Generalized Spectral Clustering : Code related to [arxiv.org/abs/2203.03221](arxiv.org/abs/2203.03221) presenting a new directed graph learning framework, applied to spectral clustering.
- Colles ECG2 : A collection of undergraduate mathematics exercises (in French), given to second year students in elite _classe préparatoire_ french curriculum.
- Femur Modeling : A computational anatomy project on human femur data. Includes a flexible [LDDMM](https://en.wikipedia.org/wiki/Large_deformation_diffeomorphic_metric_mapping) Python implementation, a powerful (and beautiful) algorithm leveraging Riemannian Embeddings of shapes.
- dotfiles : (nearly) Everything needed to build my ubuntu config, based on i3, alacritty, fish, nvim and zen. Includes a neat [Typst](typst.app) integration in Neovim.
